Conceptos de Git

Git Init

Git Init inicializa un nuevo repositorio Git en un directorio. Debes ejecutar este comando una vez al comienzo del proyecto.

Git Clone

Git Clone clona un repositorio Git existente desde una URL a tu sistema local. Es útil para descargar un proyecto desde un repositorio remoto.

Git Add [archivos]

Git Add [archivos] agrega cambios al área de preparación (staging) para que estén listos para ser confirmados. Puedes especificar archivos individuales o usar "." para agregar todos los archivos modificados.

Git Commit -m "Mensaje"

Git Commit -m "Mensaje" crea una confirmación (commit) con los cambios en el área de preparación junto con un mensaje que describe los cambios realizados.

git status

git status muestra el estado actual del repositorio, incluyendo archivos modificados, en el área de preparación y sin seguimiento.

Git Log

Git Log muestra un registro de confirmaciones que proporciona información sobre las confirmaciones anteriores, incluyendo el autor, la fecha y el mensaje del commit.

Git Branch

git branch muestra una lista de ramas (branch) en el repositorio y resalta la rama en la que te encuentras actualmente.
